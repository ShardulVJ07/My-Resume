Projects

AgriDrone -Agricultural Quad

Built an Agricultural Aviation system using BLDC motors, controllers, and feedback sensors. cost-effective solutions for farmers, integrating robust control systems, communication modules, and hardware components. Designed custom PCBs for motor control, and implemented power distribution circuits, and buffer boards for sensor data.

Smart Keychain

Working on Product based Project a Smart Keychain with a compact PCB design utilizing ESP12E Wroom for wireless connectivity. Integrated a CH340 autoreset circuit for seamless USB communication, LDO regulators for stable voltage supply, and a rechargeable battery system for portability.

Smart Device for Visually Impaired People - Patented

Created a device with computer vision and accelerometer-based object detection. Utilized ESP32 for PCB design and manufacturing, integrating location tracking and alert systems for enhanced user safety. Designed power-efficient circuits, custom shields for connectivity, and buffer boards to manage sensor inputs and communication modules.

Websocket Based Model for Remote System

Developed a system for efficient city/village garbage management. Utilized AWS for remote WebSocket communication, ReactJS for the interface, and ESP32 for programming. Designed and manufactured PCBs with robust power circuits, and used buffer boards for integration of sensors and communication modules. .


VLSI-Based Design of ALU

Designed and simulated a VLSI-based Arithmetic Logic Unit (ALU). Focused on transistor-level circuit design, layout optimization, and thorough testing for performance and reliability.

CV-Based Robotics Locomotion Algorithm using ROS

Designed a robot with self-made microcontroller boards and Raspberry Pi. Implemented computer vision and mechanical systems with 3D-printed cases.
